# Microsoft Password Expiration Email Templates

Professional HTML email templates replicating Microsoft's official password expiration notification design. Available in English and Turkish.

## Overview

These templates provide pixel-perfect replicas of Microsoft's password expiration reminder emails, featuring authentic Microsoft branding, responsive design, and professional styling suitable for legitimate security awareness training and testing purposes.

## Features

- **Responsive Design** - Optimized for both desktop and mobile email clients
- **Microsoft Branding** - Official color scheme and typography (Segoe UI)
- **Self-Contained** - All CSS included internally for maximum email client compatibility
- **Bilingual** - English and Turkish versions available
- **Professional Layout** - Includes header, alert boxes, CTA buttons, and footer
- **External Assets** - Microsoft logo loaded from CDN
- **Customizable** - Easy variable replacement for personalization

## Files

- `microsoft-password-expire-eng.html` - English version
- `microsoft-password-expire-tr.html` - Turkish version (Türkçe)
